# Тепловой и динамический расчёт дизельного двигателя

Программа выполняет полный тепловой и динамический расчёт многоцилиндрового
дизельного двигателя по методике БНТУ (кафедра «Двигатели внутреннего сгорания»).

На выходе: индикаторная диаграмма, графики сил в КШМ, диаграмма крутящего
момента, расчёт поршня, а также DXF-чертежи для Компас-3D / AutoCAD.

---

## 1. Назначение программы

Скрипт `main.py` автоматически:

- подбирает недостающие параметры рабочего цикла (k, n₂, Tr),
- рассчитывает давления и температуры во всех точках цикла,
- определяет основные размеры двигателя (D, S),
- вычисляет индикаторные и эффективные показатели,
- строит индикаторную диаграмму,
- выполняет динамический расчёт КШМ (силы Pг, Pj, PΣ, K, T),
- строит диаграмму крутящего момента,
- рассчитывает поршень на прочность.

## 2. Инструкция по запуску

### Требования

- **Python** версии 3.8 или выше.
- Около 50 МБ свободного места (библиотеки + виртуальное окружение).

### Шаг 1. Установите Python

- **Windows**: скачайте с [python.org](https://www.python.org/downloads/)
  (при установке отметьте «Add Python to PATH»).
- **Linux (Ubuntu/Debian)**:
  ```bash
  sudo apt install python3 python3-venv python3-pip
  ```
- **Linux (Arch)**:
  ```bash
  sudo pacman -S python python-pip
  ```
- **macOS**: `brew install python` или скачайте с python.org.

### Шаг 2. Откройте терминал / командную строку

Перейдите в папку с программой:

```bash
cd /путь/к/папке/ddddddd
```

### Шаг 3. Запустите скрипт установки

| Система  | Команда                               |
|----------|---------------------------------------|
| Windows  | дважды кликните `run.bat`             |
| Linux    | `bash run.sh`                         |
| macOS    | `bash run.sh`                         |

Скрипт сам создаст виртуальное окружение, установит библиотеки и запустит расчёт.

### Альтернативный ручной запуск

```bash
# 1. Создать виртуальное окружение
python3 -m venv .venv

# 2. Активировать
# Linux/macOS:
source .venv/bin/activate
# Windows:
.venv\Scripts\activate.bat

# 3. Установить библиотеки
pip install -r requirements.txt

# 4. Запустить
python main.py
```

## 3. Как заполнять исходные данные

Все параметры задаются в файле **`input.txt`**. Формат строки:

```
Имя = Значение     # комментарий (всё после # игнорируется)
```

Примеры допустимых строк:

```
Ne = 475           # мощность в кВт
epsilon = 17       # степень сжатия
alpha = 2.0        # коэффициент избытка воздуха
```

### Таблица параметров

| Параметр | Описание | Ед. изм. | Диапазон / значение |
|----------|----------|----------|---------------------|
| `Ne` | Эффективная мощность | кВт | берётся из задания |
| `epsilon` | Степень сжатия | — | 13…22 |
| `n` | Частота вращения коленвала | об/мин | берётся из задания |
| `i` | Число цилиндров | — | берётся из задания |
| `theta` | Тактность | — | 4 (4-тактный) или 2 (2-тактный) |
| `SD` | Отношение хода поршня к диаметру S/D | — | 0.8…1.3 |
| `alpha` | Коэффициент избытка воздуха (α) | — | 1.4…2.2 |
| `p0` | Давление окружающей среды | МПа | 0.1 |
| `T0` | Температура окружающей среды | К | 298 (25 °C) |
| `nk` | Показатель политропы сжатия в компрессоре | — | 1.4 |
| `incps` | Потери давления в воздухоохладителе | МПа | 0.001…0.005 |
| `incT` | Подогрев свежего заряда от стенок | К | 5…20 |
| `gammar` | Коэффициент остаточных газов (γr) | — | 0.02…0.06 |
| `Tr` | Температура остаточных газов (ориентировочно) | К | 600…1000 |
| `Hu` | Низшая теплота сгорания топлива | кДж/кг | 42500…44500 |
| `gc` | Массовая доля углерода C | — | 0.85…0.87 |
| `gh` | Массовая доля водорода H | — | 0.13…0.15 |
| `go` | Массовая доля кислорода O | — | 0…0.004 |
| `n1` | Показатель политропы сжатия (n₁) | — | 1.32…1.37 |
| `k` | Степень повышения давления (λ = pz/pc) | — | 1.2…2.5 |
| `n2` | Показатель политропы расширения (n₂) | — | 1.18…1.28 |
| `Tz` | Температура конца сгорания | К | 1700…2300 |
| `phip` | Коэффициент полноты индикаторной диаграммы (φп) | — | 0.90…0.96 |
| `num` | Механический КПД (ηм) | — | 0.75…0.90 |
| `lambda_k` | Отношение радиуса кривошипа к длине шатуна (λ) | — | 0.25…0.32 |
| `m_j` | Масса возвратно-поступательно движущихся частей | кг | 2.0…5.0 |

### Важно

- **Всегда** проверяйте, что gc + gh + go ≈ 1.0.
- Если ввести нереалистичные значения (например, epsilon = 50),
  программа может выдать численно корректный, но физически бессмысленный
  результат. Придерживайтесь рекомендованных диапазонов.
- Параметры `k`, `n2`, `Tr` будут автоматически уточнены оптимизатором
  при запуске расчёта.

## 4. Где искать результаты

### Консольный вывод

При запуске в терминал выводится:

- таблица с входными данными,
- пошаговый расчёт с формулами,
- проверка температуры остаточных газов и мощности,
- динамическая таблица сил с шагом 30°,
- результаты расчёта поршня.

### Папка `svg/` — векторные диаграммы

Файл | Что содержит
-----|-------------
`indicator_diagram.svg` | Индикаторная диаграмма (p–V)
`dynamic_forces.svg` | Графики сил Pг, Pj, PΣ, T, K
`torque_diagram.svg` | Диаграмма крутящего момента

Совет: SVG без потери качества вставляется в LibreOffice Writer, Word
или LaTeX (через `\includesvg` из пакета `svg`).

### Папка `dxf/` — чертежи для САПР

Файл | Что содержит
-----|-------------
`indicator_diagram.dxf` | Индикаторная диаграмма (с ключевыми точками a, c, z', z'', b, r)
`dynamic_forces.dxf` | Диаграммы сил Pг, Pj, PΣ, T, K
`torque_diagram.dxf` | Крутящий момент (суммарный + от каждого цилиндра)
`sheet_a1.dxf` | Компоновка всех диаграмм на листе А1 по ГОСТ/ЕСКД

Масштабы μp (МПа/мм) и μV (л/мм) зафиксированы и указаны в чертеже.
DXF-файлы совместимы с **Компас-3D**, **AutoCAD**, **LibreCAD**, **SolidWorks**.
